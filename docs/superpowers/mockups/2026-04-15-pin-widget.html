Pin widget — variants

How a pinned todo should render in the widgets section of the home page. A1/A2 = Option A (each pinned todo = its own tile). B1/B2 = Option B (one combined "Pinned" widget).

Each mockup shows the whole home page above the widgets section so you can see scale, spacing, and how pinned todos mix with existing habit/timer/alarm widgets.

A1 · Per-todo tile, minimal Option A

Each pinned todo = one 164×156 widget. Thin pink top-edge, checkbox + title + deadline. Feels like the existing habit/timer tiles.
